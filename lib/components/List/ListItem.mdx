# ListItem

A `ListItem` represents a piece of content; a page, person or an object.

<Primary />

## How to use

```tsx
import { ListItem } from "@altinn/altinn-components";
import { TeddyBearIcon } from "@navikt/aksel-icons";

<ListItem
  title="Title"
  icon={{ theme: "surface", svgElement: TeddyBearIcon }}
  badge={{ theme: "subtle", label: "2 av 3" }}
  linkIcon
/>;
```

## Using icons or avatars

Add an icon using the `icon` prop. Icons can be references by name or using IconProps, making it possible to add an optional `color` and `theme`.

<Canvas of={ListItemStories.IconTypes} />

Add an avatar using the `avatar` prop.

<Canvas of={ListItemStories.Avatars} />

Add an avatar group using the `avatarGroup` prop.

<Canvas of={ListItemStories.AvatarGroups} />

If you need a custom icon you can override it by adding your own component as `icon`:

<Canvas of={ListItemStories.OverridingIcon} />

## Using labels

By default labels are built up of a `badge` and an optional `description`. You can override both using the `label` prop:

<Canvas of={ListItemStories.OverridingLabel} />

## Collapsible/expanded

Make items collapsible by adding the `collapsible` prop. Adding the `expanded` prop will expose the `children` of the list item.

<Canvas of={ListItemStories.Collapsible} />

## Arrow and badges

Use the `linkIcon` prop to indicate to the user that your item is a link. The `badge` prop is also useful for informing the user of what to expect when interacting with the item. For example if the item is collapsable.

<Canvas of={ListItemStories.Badges} />

Badge can be replaced with your own component unsing the `badge` prop:

<Canvas of={ListItemStories.OverridingBadge} />

## Loading state

Add the `loading` prop to represent that a list item is loading .

<Canvas of={ListItemStories.LoadingState} />

## Custom controls

Add custom controls using the `controls` prop.

<Canvas of={ListItemStories.CustomControls} />

When adding controls, list item interaction is split from controls and default badges are moved next to the label.

Make items selectable by using the `select` prop. Add `selected` to select the whole element.
