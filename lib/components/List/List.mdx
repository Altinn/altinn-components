# Using lists

Use lists to represent links to spesific objects, for example a list of people or a list of services.

Build lists using `List` and `ListItem`, or use `List` with `items` for simple, unified lists.

```tsx
import { List, ListItem } from "@altinn/altinn-components";

return (
  <List>
    <ListItem />
    <ListItem />
    <ListItem />
  </List>
);
```

While lists can be modified to represent spesific needs, we should try to establish common patterns for similar tasks across Altinn.

## How to use

```tsx
import { ListItem } from "@altinn/altinn-components";
import { ChevronRightIcon, TeddyBearIcon } from "@navikt/aksel-icons";

<ListItem
  title="Title"
  icon={{ theme: "surface", svgElement: TeddyBearIcon }}
  badge={{ theme: "subtle", label: "2 av 3" }}
  linkIcon
/>;
```
