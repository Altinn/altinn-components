# Using lists

Use lists to represent links to spesific objects, for example a list of people or a list of services.

Build lists using `ListBase` and `ListItem`, or use the `List` component for simple, unified lists.

```tsx
import { ListBase, ListItem } from "@altinn/altinn-components";

return (
  <ListBase>
    <ListItem />
    <ListItem />
    <ListItem />
  </ListBase>
);
```

## ListItem

A `ListItem` represents a piece of content; a page, person or an object.

<Canvas of={ListItemStories.Default} />

## How to use

```tsx
import { ListItem } from "@altinn/altinn-components";

<ListItem
  title="Title"
  icon={{ theme: "surface", name: "teddy-bear" }}
  badge={{ theme: "subtle", label: "2 av 3" }}
  linkIcon="chevron-right"
/>;
```

## Collapsible/expanded

Make items collapsible by adding the `collapsible` prop. Adding the `expanded` prop will change the default `linkIcon` and expose the `children` of the list item.
