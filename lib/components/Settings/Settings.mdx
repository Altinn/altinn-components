# Settings

Use `SettingsSection`, `List` and `SettingsItem` to build settings sections.

```tsx
return (
  <SettingsSection>
    <List>
      <SettingsItem />
      <Divider as="li" />
      <SettingsItem />
      <Divider as="li">
      <SettingsList items={[items]}>
    </List>
  </SettingsSection>
);
```
