# Dialog metadata

Dialog metadata should include a status and timestamp, and may contain additional metadata to indicate due date, number of attachments. Number of outgoing and/or incoming transmissions is presented next to the status.

<Canvas of={DialogMetadata.InProgress} />

<Canvas of={DialogMetadata.Completed} />
