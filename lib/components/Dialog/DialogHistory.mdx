# Dialog history and logs

A dialog can include logs representing events that have happened in the past.

## Activity logs

The activity log is a list of every activity related to the dialog, ordered by time.

## History

The history is a curated list of activities and transmissions, informing the user of important events that has led to the current state of the dialog.
